## Keeping This Updated

### For Claude Code Users

When working with Claude Code on this repository:

1. **Use Claude Code slash commands** to streamline your workflow:
   - Reference existing scripts and patterns in your implementation
   - Ask Claude to explain existing scripts before modifying them
   - Use the file explorer to navigate the repository structure

2. **Before making changes**:
   - Read existing scripts to understand patterns and conventions
   - Ask Claude to review changes before committing
   - Test scripts locally before pushing

3. **Keep documentation current**:
   - Update this file if you add new script categories or patterns
   - Add README files to subdirectories with complex logic
   - Include usage examples in comments for non-obvious scripts

4. **Commit guidelines**:
   - Use clear, descriptive commit messages
   - Reference what scripts were added, modified, or removed
   - Keep commits focused on logical units of work
